Thank you for your interest in contributing to SkillVerse!
SkillVerse is a modern, frontend-focused E-learning platform built to simulate a real SaaS product.

## Ways to Contribute
- Improve UI/UX or animations
- Add new course content or quizzes
- Improve performance or accessibility
- Enhance gamification or analytics
- Improve documentation

## Getting Started
1. Fork the repository
2. Clone your fork locally
3. Create a new branch:
   git checkout -b feature/your-feature-name
4. Make your changes
5. Commit with clear, meaningful messages
6. Push your branch and open a Pull Request

## Code Guidelines
- Follow existing React & TypeScript patterns
- Keep components reusable and clean
- Avoid adding backend services (LocalStorage-first design)

## Issues
Please check existing issues before opening a new one.
Comment on an issue if you want to work on it.
